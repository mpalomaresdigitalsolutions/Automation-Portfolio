# speckit.constitution

Creates or updates the project's governing principles and development guidelines.

## Usage
`/speckit.constitution [description]`

## Description
This command creates or updates the project constitution in `.specify/memory/constitution.md`. The constitution defines the core principles, development guidelines, and project-specific rules that will guide all subsequent development.

## Examples
- `/speckit.constitution Create principles focused on code quality, testing standards, user experience consistency, and performance requirements`
- `/speckit.constitution Update constitution to include new security guidelines`

## Output
Creates/updates `.specify/memory/constitution.md` with the specified principles and guidelines.
